** CSS에서 요소 => BOX => block,inline
	
* box의 정렬
   block box  => 세로정렬 
   inline box => 가로 정렬 

* box의 모양
   block box => 부모box의 넓이 만큼 차지
   	        width, height 사용가능
	        자신안에 블록박스나 인라인박스 모두 담을 수 있음 

   inline box => 자신안에 있는 내용(box)만큼 차지
	         width, height 사용 불가	         
	         위(top),아래(bottom) 마진사용불가
	         자신안에는 inline박스만 담을 수 있음(글자는 inline취금..합니다..)

* block박스를 가로정렬 => float나  inline-block 사용
